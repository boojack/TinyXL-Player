# A Tiny Video Player

## What is it

There are lots of unhappy usage in the SPlayer, so here we go to make a Video Player.

This is an absolutly TINY Video Player which just made during 1 hour!

## How to use it

1. Download this project source code;
2. Run these command:

``` bash
yarn install
# For tester
npm start
# For user
npm run package
# or
npm run make
```
